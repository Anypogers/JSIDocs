# JSI File Structure

The project structure goes as follows:
```
just-simple-intents/
├── core/
│   ├── intentManager.js
│   ├── intentRouter.js
│   └── config.js
│
├── intents/
│   ├── foo/
│   │   ├── index.js
│   │   ├── training.json
│   │   └── README.md
│   └── bar/
│       ├── index.js
│       ├── training.json
│       └── README.md
│
├── index.js
├── package.json
└── README.md
```
